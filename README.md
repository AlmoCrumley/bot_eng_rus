English-Russian Learning Bot 🤖📚
A Telegram bot designed to help users learn English and Russian vocabulary

Features
Bilingual Dictionary: Manage word pairs between English and Russian

User Progress Tracking: Monitors learning progress (implied from database/ folder)

Customizable Study Sets: Create and manage your own word lists


bot_eng_rus/
├── config_data/       # Configuration files
│   └── config.py      # Bot token and settings
├── database/          # Database connection and models
├── db_bot/            # Database operations for the bot
├── handlers/          # Command and message handlers
├── keyboards/         # Inline and reply keyboards
├── .idea/             # IDE project files (ignore)
├── main.py            # Entry point for the bot
└── requirements.txt   # Python dependencies
